## 🧮 Funções implementadas

### 🔹 soma(a, b)

Retorna a soma de dois números.

### 🔹 media(...nums)

Recebe vários números e retorna a média entre eles.

### 🔹 maximo(a, b)

Retorna o maior valor entre dois números.

### 🔹 fatorial(n)

Calcula o fatorial de um número.

---

## 📌 Diferença entre Parâmetro e Argumento

### 🧩 Parâmetro

É a variável definida na função.

```javascript
function soma(a, b) {
    return a + b;
}
```

* `a` e `b` são **parâmetros**

---

### 🎯 Argumento

É o valor que você passa ao chamar a função.

```javascript
soma(10, 5);
```

* `10` e `5` são **argumentos**

---

## 🧠 Resumindo

* **Parâmetro** → variável dentro da função
* **Argumento** → valor enviado para a função
